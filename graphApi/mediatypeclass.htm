MediaType Class

"Media types" used to be known as MIME types.  Values are defined by RFC 2046 and registered by IANA.

Equivalent to: dcmi:MediaType

Enumerated Value

Description

text/html

Text formatted with HTML markup tags.

audio/mp4

An audio clip formatted in MP4.

image/jpeg

An image formatted in JPG.

application/json

An object value represented as text in JSON format.

application/vnd.ms-excel

An Excel spreadsheet.

etc...

... other values defined by IANA
