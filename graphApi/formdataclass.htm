FormData Class

When someone fills out a Form, the results are captured in a FormData object.  The “form” property indicates which form the data is for.   The "data" property contains the data.  Data values are correlated to the form by matching the form data element’s name to a property name in the data object..

 

Property

Datatype

Req

Description

form

Form

MUST

The data in this formData object is in response to the fields in the form specified by this property.

data

PlainObject

MUST

 

The raw data.  This is a plain object whose properties correspond to field names in the form.  All of the properties in this object correspond to a named form field.
