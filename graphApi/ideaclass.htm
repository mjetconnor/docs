Idea Class

Concepts: Idea Concepts

An explicit description of an invention or problem solution with the intention of implementation as a new or improved product, service, or process within an organization.  (Riedl et al)

The primary purpose of the Mindjet System is to manage, organize, evaluate and implement ideas.

Many commonly used properties for ideas are defined by the inherited Entity class, such as teams, discussions, followers and assessments.

Governor

The governor that applies to this idea.  The governor defines how the evolution of the idea is managed, such as its stages and graduation rules.  The governor is set when the idea is created.

The inquiry to which the idea was originally posted.

The idea's current stage of development.

uiForm

Form

MAY

The uiForm that applies to this idea.  This informs displays of this idea as to how non-standard properties (aka “templated properties”) should be rendered in displays or prompt forms.

IdeaScore

The collection of all ideaScores whose base is this idea.  In other words, if this property exists:
   [ scoreX base ideaY ]
then this property exists:
   [ ideaY score scoreX ].
