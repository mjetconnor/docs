Reference Class

Describes a thing that is used as a reference in the description or documentation of another thing.

 

Property

Datatype

Req

Description

resource

Thing

MUST

The substance of the reference.  Common values are MediaResources that represent document or image files or web pages.  They could also be other MindjetGraph elements such as ideas or comments.

weight

Decimal

MAY

An optional weight may be specified that represents the importance of this reference.  “Normal” is a value of 100.

role

Thing

MAY

Something that describes the value or purpose of this reference.

A common value to specify as the role is a value of class ReferenceRole.  If no value is specified for this property, it is reasonable to assume a value of ReferenceRole.inSupportOf.

inSupportOf

inSupportOfs

Thing

[ * ]

The things that this reference is in support of.
