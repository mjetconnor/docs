UserScore Class

A facet used for tracking status information about an Agent.

Subclass of: Facet

 

Property

Datatype

Req

Description

base

Agent

MUST

The Facet class defines the base property.  For UserScores, this MUST be an Agent  This is not strictly limited to User entities, but users are the most common type of agent that has scores.

context

Thing

MUST

The Facet class defines the context property.  

reputation

Decimal

MAY

A number that represents the reputation of the user within this context.  A higher value indicates that the user is held in higher esteem.

points

Decimal

MAY

The user's game points balance within this context.
