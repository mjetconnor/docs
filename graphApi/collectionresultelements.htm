Collection Results Elements

Collection values are represented in responses as objects that have these elements:

Element

Required

Description

data

yes

The value of this element is an array whose elements are elements of the collection.

paging

no

If included, this element's value is an object that has two optional elements: next and previous.  The values of these elements are the URLs to use to retrieve the next or previous page of elements from the collection.

totalSize

no

totalSize SHOULD be included if the server can reasonably compute its value without undue cost.  If this element is not included, it may be inferred that computation of the total size would be expensive.


Example:

In this example, the communities property is a part of a larger response.  Its value is a collection.  Here we see the first three elements in the collection, all Community entities, and that there are 16 total elements in the collection.  To get the next block of entries in the collection, execute an HTTP GET request using the URL "https://www.example.mindjet.com/api/ds1/tenant-1/communities?after=2".
