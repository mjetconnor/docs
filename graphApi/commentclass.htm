Comment Class

Concepts: Comment and Forum Concepts

A comment in a forum.  In general, comments may be made in a broad variety of sioc:Forum’s.  Mindjet comments are generally made to discussions that are usually associated with an entity that is the “object” of the discussion.

Subclass of: sioc:Comment

Subclass of: sioc:Post

Property

Datatype

Req

Description

forum

Discussion

SHOULD

The forum in which this comment was made.  For comments made in Mindjet, the forum will be a Discussion.

This property is equivalent to “sioc:has_container”.

content

String

SHOULD

The text of the comment.

replyTo

Comment

MAY

If the comment is made in reply to another comment, this property indicates which comment that was.  This property is the inverse of “sioc:has_reply”.

hasReply

hasReplies

Comment

RULE

The comments that have been made in reply to this comment.  This property is the inverse of the replyTo property.  In other words, if this property exists:
   [ commentA replyTo commentB ]
then this property exists:
   [ commentB hasReply commentA ]
