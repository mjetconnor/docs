Access Token Response

There is a common structure to the response message that include a requested access token.  This information may be returned in different formats, such as url-encoded in a redirect URL or as JSON in the body of a POST response.

 

Element

Description

access_token

The requested access token.

refresh_token

A token that may be used to get a new access token when this one times out.

This value is omitted from the response to grant_type=password requests.

issued_at

The time when the access_token was issued.  It is represented as a 13 digit integer that is a Unix-time value in milliseconds,

userId

The entity ID of the user authenticated by this access token.

communityId

The entity ID of the home community, in the context of the authorization service URL that was used to request this access token.

state

If a state parameter was specified in the authorization request, it is echoed back here.
