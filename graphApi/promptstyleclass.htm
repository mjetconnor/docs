PromptStyle Class

A value that informs a display application as to how best to prompt for a value.  A prominent use of this value is as part of a FormField object that describes the fields in a form.

Enumerated Value

Description

radioButton

If possible, prompt using radio buttons.

checkBox

If possible, prompt using checkboxes.

singleSelection

If possible, prompt using a single-selection menu.

multipleSelection

If possible, prompt using a multiple-selection menu.

richText

If possible, prompt  using a rich-text editor such as TinyMCE.
