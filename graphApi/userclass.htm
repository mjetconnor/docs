User Class

Concepts: Agent Concepts

A person that has a login account in the Mindjet system.

Note that every User has exactly one account that describes that user's access.  But a given account might relate to multiple User records.  In this case, each User instance represents a different profile, possibly with different display names and avatars, all with the same underlying access and authority.
