Plural and Singular Property Names

When property names appear in a query URL, it matters if the singular or plural form of the name is used.  If the singular form of the name is used, the resulting value will be a scalar value, not a collection.  A conversion is performed if necessary.  Conversely, if the plural form of the name is used, the resulting value will be a collection.  Again, a conversion is performed if necessary.

If the base value is a collection, but the request was made using the singular form of the property name, the "primary" element of the collection is used.  In most cases, this is the first entry according to the collection's natural order.

If the base value is not a collection, but the request was made using the plural form of the property name, a temporary collection is made that has exactly one element whose value is the base value.

See Singular / Plural Property Names for a complete mapping of the singular and plural forms of property names.
