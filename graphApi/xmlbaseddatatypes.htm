XML-Based Datatypes

These datatypes correlate directly to types defined for XML by the XSD standard  [ http://www.w3.org/TR/xmlschema-2/ ].

MindjetGraph Datatype

Equivalent XML Datatype

Description

String

xsd:string

Typical free-text value that may contain any content, including all kinds of HTML markup

Integer

xsd:integer

An integer value.  For the purposes of this specification, integers do not have size limitations, akin to "decimal" types in SQL.

Decimal

xsd:decimal

A number.  Similar to Integer but with the addition of a scale, which means that there may be  fractional part.

Boolean

xsd:boolean

A boolean value: true or false.

DateTime

xsd:dateTime

A date-time value with a precision of 1 millisecond.

Date

xsd:date

A date value with no time component.

Time

xsd:time

A time value with no date component.

URI

xsd:anyURI

A URI value.  URLs are a kind of URI.

SimpleText

xsd:token

A free-text value that contains no HTML markup or rich text elements.  It is used to identify text values that will never have rich text features or embedded mark-up elements.  The SimpleText datatype should be used sparingly.  It should be used instead of String only if the use of rich text elements is inherently inappropriate to the value.

This datatype does not directly correspond to any XML type.  Its base class is xsd:token.  xsd:string is too broad, and xsd:Name is too limited in that that type imposes a limit on the character set in a way that SimpleText does not.
