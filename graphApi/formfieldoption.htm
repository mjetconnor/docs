FormFieldOption Class

When a FormField has an enumerated set of possible values, as indicated by its options property, those values are described by objects of this type.

 

Property

Datatype

Req

Description

value

SimpleText

MUST

The value of the option.

label

String

MAY

The display label for the option, if it is different from the value.

selected

Boolean

MAY

True if this option should be selected by default in a display.

 

See FormField class.
