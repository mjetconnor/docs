ActivityLayout Class

Concepts: Activity Concepts

Activity layouts provide information to applications that handle activity events as to how they should be displayed. Get the set of possible layouts from the Verb of the activity event.  Then pick the layout that most closely matches the event in terms of the object, target and actor types specified in the layout.  From that layout, choose the template that best corresponds to the application's display or usage.
