RDF and OWL Datatypes

These datatypes correlate to types defined by RDF and OWL.

MindjetGraph Datatype

Equivalent Datatype

Description

Resource

rdfs:Resource

Any object that has a persistent identity.  This is very much like Mindjet:Entity but broader, in that it includes objects whose system-of-record is not a SpigitEngage platform.

Class

rdfs:Class

A datatype value.

Thing

owl:Thing

Any kind of object value without restriction, from simple literal numbers and text strings to complex entities.  This is broader than Resource because literal values are included.
