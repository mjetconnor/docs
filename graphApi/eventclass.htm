Event Class


Property

Datatype

Req

Description

eid

URI

MUST

A unique identifier for the event.  This value is guaranteed to be unique for all Mindjet events everywhere.  For events that are also entities, this is the entity ID.

type

Class

MUST

This event object’s class.  It is probably something more specific than “Event”, such as “Mindjet:ActivityEvent” or “Mindjet:EntityChangeEvent”.

date

DateTime

MUST

The time that the event occurred.

dataSpace

DataSpace

MUST

The dataspace most closely associated with the event.
