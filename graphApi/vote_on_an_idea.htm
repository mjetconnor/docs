Vote on an Idea

Vote on an idea by posting a vote to that idea.

POST https://www.example.mindjet.com/api/ds1/node-89523  

with headers:

Content-Type: application/json

Authorization: Bearer 914a9b4d83fa7333d3b517643a30bac2

and request body:

{

   type:"Vote",

   value:"1",

}

gets the response:

{

 "version": "1.0",

 "query": "POST https://www.example.mindjet.com/api/ds1/node-89523",

 "xmlns": {

   "sief": "http://purl.org/sief/2013/10/",

   "_ds": "https://www.example.mindjet.com/api/ds1/",

   "_host": "https://www.example.mindjet.com/api/"

 },

 "results": [

   {

     "type": "Vote",

     "benchmark": {

       "type": "UpDownBallot"

     },

     "object": {

       "eid": "_ds:node-89523",

       "type": "Idea"

     },

     "valuation": 1

   }

 ]

}

 


This also is a good example of a response value that is not an entity.  The response is an object described by the Vote class.  But it does not have an eid property, so it is not an entity
