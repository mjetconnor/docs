Posts to Discussion Entities

Post requests are handled as defined here when the target is an entity of type Discussion.  The appropriate response is determined by the action and type post properties.

Post a comment

Post property

Datatype

Req

Description

action

String

MAY

"post" or undefined

type

String

MUST

"Comment"

content

String

MUST

The text of the comment.

Returns:  The created Comment entity.
