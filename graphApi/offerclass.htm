Represents an offer that may be accepted.

Offers are usually Entities, but not required to be so.


Property

Datatype

Req

Description

creator

Agent

MUST

The agent who made the offer.  (Same property as defined for Entity class)

created

DateTime

MUST

The time when the offer was made.  (Same property as defined for Entity class)

title

SimpleText

SHOULD

The display label of this offer.

description

String

MAY


object

Thing

MAY

The thing being offered.

bid

   bids

Bid

RULE

Any bids made in relation to this offer.  This is the inverse of the offer property on Bids.   In other words, if this property exists:
   [ bidA offer offerB ]
then this property exists:
   [ offerB bid bidA ]

acceptance

   acceptances

Acceptance

RULE

Any acceptances related to this offer.  This is the inverse of the offer property on Acceptances.   In other words, if this property exists:
   [ acceptanceA offer offerB ]
then this property exists:
   [ offerB acceptance acceptanceA ]
