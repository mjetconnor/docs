Country Class

ISO 3166 country codes.  Note that to distinguish these values from ISO-639 language codes, which often have similar values, country codes are typically upper-case.

This table shows a few illustrative values and is not exhaustive.  See http://www.iso.org/iso/country_codes for the current complete list of values.
