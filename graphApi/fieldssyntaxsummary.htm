Field Syntax Summary

fieldsQueryParam

=

"fields=" fieldList

fieldList

=

fieldPhrase [ "," fieldPhrase ... ]

A field list is a comma-delimited list of field phrases.

fieldPhrase

=

propertyExpr | macroName | macroDef

A field phrase can be any one of a property expression, a macro name or a macro definition.

propertyExpr

=

propertyName   [ ":" ( "[" fieldList "]" | macroName ) ]   [ "*" pageSize ]

A property expression is a property name,
an optional field list after a colon and surrounded by square brackets,
and an optional page size after an asterisk.

propertyName

=

name

pageSize

=

integer

macroName

=

"@" name

macroDef

=

macroName "=[" fieldList "]"
