FormField Class

Description of a field within a Form.

Property

Datatype

Req

Description

name

SimpleText

SHOULD

The name of the field, for the purpose of labelling the values in a corresponding FormData.  This is not the display text for the field.  Use the title for that.

title

String

SHOULD

The display label to be used when rendering this field.

fieldType

Class

SHOULD

When a value is given for this field, for instance in a corresponding FormData object, its value should be consistent with this type.

position

Decimal

SHOULD

The ordinal position of this field within the overall form.

required

Boolean

MAY

True IFF data entry into the form is not complete without a value for this field.  Default: false.

description

String

MAY

 

promptStyle

PromptStyle

MAY

Additional information needed to render the appropriate kind of input element for this field.

units

Thing

MAY

If fieldType is Money, this could be a Currency value.

If fieldType is Interval, this could be a TimeUnit value.

option

options

FormFieldOption

[ * ]

When a field has an enumerated set of acceptable values, those values are described here.


Common values for the "fieldType" property, and how they typically apply to values in a form:


"fieldType" Class value

Description of value expected to be entered for the field

SimpleText

A single line of text, typically comparatively short.

String

Longer text that could span to multiple lines.  A promptStyle value of “richText” indicates that a rich text editor such as TinyMCE should be used if possible.

Integer

An integer value.

Decimal

A number that may include a fractional part.

Boolean

True/False

DateTime

A date/time value; a timestamp.

Date

A date value with no time component.

Time

A time value with no date component.

User

A SpigitEngage user.
