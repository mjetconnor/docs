IdeaScore Class

A facet used for tracking the metrics and status information related to an idea.

Take special note that all of these properties, except for "base", are marked only that they MAY be provided.  This values tend to be tightly related to specific application usages, implementations and configurations.

Subclass of: Facet

Property

Datatype

Req

Description

base

Idea

MUST

The Facet class defines the base property.  For IdeaScores, this MUST be an Idea.

context

Thing

MUST

The Facet class defines the context property.

upVotes

Integer

MAY

Number of up votes made against this idea, in the manner of a LikeBallot or UpDownBallot.

downVotes

Integer

MAY

Number of down votes made against this idea, in the manner of an UpDownBallot.

viewCount

Integer

MAY

Number of views made of this idea.  The definition of "view" depends on application details.

commentCount

Integer

MAY

The number of comments made, including replies, with this idea as the topic.

approvalRating

Decimal

MAY

A metric that digests the vote counts against this idea, as compared to votes against related ideas, into a single number.  The exact algorithm used to calculate this number may vary, for instance across tenants, communities or inquiries.  It should at least be consistent across a related set of ideas, which in particular means it should be consistent at least within a given inquiry.

starVotes{X}

Integer

MAY

A set of properties where the suffix of the property name - the {X} - indicates a particular star vote value.  A typical star voting scheme uses a rating range of 1 to 5.  The associated ideaScore properties would then be named:

starVotes1
starVotes2
starVotes3
starVotes4
starVotes5

The value of each of these properties is the total count of votes at that star rating.

predictionResult

FormData

MAY

added v3.7

If the idea has received prediction votes, this property will be included.  In typical fashion for FormData objects, the elements of the data property correspond to the fields defined by the form.  The data elements are objects that contain these properties:

mean The current prediction value for this property for this idea.
deviation A measurement of the narrowness of the range of prediction.
