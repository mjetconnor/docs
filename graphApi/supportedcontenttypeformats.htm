Supported Content-Type Formats

For HTTP queries that have bodies, which is to say PUT and POST queries, these are the supported content types:

Content type

Example of body

application/json

{ prop1 : valueA, prop2 : valueB, prop3 : valueC }

application/x-www-form-urlencoded

prop1=urlEncodedValueA&prop2=urlEncodedValueB&prop3=urlEncodedValueC
