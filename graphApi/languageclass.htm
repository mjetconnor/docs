Language Class

ISO 639-2 and 639-1 country codes.  Note that to distinguish these values from ISO-3166 country codes, which often have similar values, language codes are typically lower-case.

This table shows a few illustrative values and is not exhaustive.  See http://www.iso.org/iso/home/standards/language_codes.htm  for the current complete list of values.
