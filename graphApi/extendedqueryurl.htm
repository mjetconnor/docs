Extended Query URL

There are elements that may be added to the basic entity URI, to create a request URL that performs simple selections.  The full request URL syntax can be summed up this way:

entityURI [ / propertyName [ ; matrixParameter ]... ]... [ ? queryParams ]

(Remember that entity IDs are URIs.  They are the same thing.)

As always, the structure of the response is the same.  The only thing that's changing is what value has been selected to be returned.
