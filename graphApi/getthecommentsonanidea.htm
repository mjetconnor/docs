Get the Comments on an Idea

In MindjetGraph, comments are organized into Discussions.  In theory, anything can have any number of discussions about that thing.  In practice, SpigitEngage only supports a single discussion for each idea.  The idea's discussion property gives you the entity that represents the comment thread.  That entity has a comments property that lists all of comments.

Let's select the discussion property of the idea we were just looking at in the previous tutorial page:

GET https://www.example.mindjet.com/api/ds1/node-75494/discussion

Most of these properties are really just bookkeeping.  The substance is all in the comments.  Let's take a closer look at them:

GET https://www.example.mindjet.com/api/ds1/node-75494/discussion/comments?fields=content,created,creator,replyTo

returns:

{

 "version": "1.0",

 "query": "GET https://www.example.mindjet.com/api/ds1/node-75494/discussion",

 "xmlns": {

   "sief": "http://purl.org/sief/2013/10/",

   "_ds": "https://www.example.mindjet.com/api/ds1/",

   "_host": "https://www.example.mindjet.com/api/"

 },

 "results": [

   {

      "data": [

        {

          "eid": "_ds:node-7093",

          "type": "Comment",

          "content": "I think your description in paragraph two could be a little clearer",

          "creator": {

            "eid": "_ds:user-37",

            "type": "User"

          },

          "created": "2014-04-02T20:40:08Z",

          "replyTo": null

        },

        {

          "eid": "_ds:node-7092",

          "type": "Comment",

          "content": "Does this idea really fit our market?",

          "creator": {

            "eid": "_ds:user-85",

            "type": "User"

          },

          "created": "2014-04-02T20:40:00Z",

          "replyTo": null

        },

        {

          "eid": "_ds:node-7094",

          "type": "Comment",

          "content": "I know of a client who could use it right now!",

          "creator": {

            "eid": "_ds:user-37",

            "type": "User"

          },

          "created": "2014-04-02T20:40:23Z",

          "replyTo": {

            "eid": "_ds:node-7092",

            "type": "Comment"

          }

        }

      ]

   }

 ]

}

Notice that the third comment, _ds:node-7094, is in reply to comment _ds:node-7092, which is the second comment.  The comments property of a discussion includes all comments in the thread, including all replies.  The "top level" comments are the ones that are not in reply to anything, in other words the ones whose replyTo property has no value or is null.
