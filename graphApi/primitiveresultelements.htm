Primitive Results Elements

In JSON, boolean and number values are represented using their native values.  For example:

{ aBooleanValue:true, anIntegerValue:1492, aDecimalValue:14.93 }

JSON does not have built-in support for dates and times.  Values for dates, times and dateTimes are included in MindjetGraph REST responses as text values formatted according to ISO 8601.  Example:

{ aDateTimeValue:"2013-11-20T13:05:45Z", aDateValue:"2013-11-20" }

Here is a brief summation of ISO 8601:

Type

ABNF

Example

Date

4DIGIT “-” 2DIGIT “-” 2DIGIT

"2013-11-20"

represents November 20, 2013

Time

2DIGIT “:” 2DIGIT “:” 2DIGIT [ “.” 1*DIGIT ]

"13:05:45.789"

represents 5 minutes after 1pm, with a millisecond part of 789

DateTime

Date “T” Time “Z”

"2013-11-20T13:05:45.789Z"

The "Z" is a shorthand for GMT.  MindjetGraph responses always use the "Z" construct.
